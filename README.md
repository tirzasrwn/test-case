# test-case

## About

Contais template for test and build code for C, C++, and Qt5 library. Qt library build with cmake.

## Usage

### C code

1. Build and run

```sh
$ gcc c-case.c -o c-case
$ ./c-case
```

### C++ code

1. Build and run

```sh
$ g++ cpp-case.cpp -o cpp-case
$ ./cpp-case
```
### Qt5 Library

1. Build and run

```sh
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./qt-case
```
